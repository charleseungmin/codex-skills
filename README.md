# Codex Skills

이 저장소는 `C:\Users\lenovo\.codex\skills` 경로에서 관리하는 재사용 가능한 Codex 스킬 모음입니다.

## 포함된 스킬

| 스킬 | 설명 |
| --- | --- |
| `code-flowchart-mermaid` | 코드 분석 결과를 바탕으로 Fragment 흐름, 화면 전환, 내비게이션 경로, 라이프사이클, 핵심 비즈니스 로직 분기를 Mermaid 플로우차트로 정리합니다. |
| `css-to-android-ui` | CSS 기반 UI 정의를 Android View XML 또는 Jetpack Compose 코드로 변환합니다. |
| `doc` | 레이아웃을 고려하면서 `.docx` 문서를 읽고, 만들고, 수정하는 작업에 사용합니다. |
| `exofit-android-conventions` | exoFit 저장소의 Android 코딩 규칙, 아키텍처 경계, 전달 절차를 따르도록 돕습니다. |
| `exofit-components-dev` | Exofit React 컴포넌트 라이브러리를 현재 구조와 공개 API에 맞게 개발하거나 수정합니다. |
| `figma` | Figma MCP 서버를 사용해 디자인 컨텍스트, 스크린샷, 변수, 에셋을 가져옵니다. |
| `figma-implement-design` | Figma 노드를 실제 서비스 코드로 옮길 때 높은 시각적 일치도를 목표로 구현합니다. |
| `gh-pr-korean` | 현재 브랜치를 푸시하고, 한글 제목과 본문을 포함한 GitHub Pull Request를 생성하거나 갱신합니다. |
| `image-to-android-ui` | UI 이미지, 목업, 스크린샷을 Android View XML 또는 Jetpack Compose 코드로 재현합니다. |
| `pdf` | 렌더링과 레이아웃을 고려해 PDF 파일을 읽고, 만들고, 검토합니다. |
| `project-convention-adapter` | 현재 저장소의 기술 스택, 빌드 도구, 아키텍처, 코딩 컨벤션을 먼저 파악한 뒤 그 프로젝트 스타일에 맞춰 구현하도록 돕습니다. |
| `screen-test-case-sheet` | 화면 단위로 QA 테스트 케이스를 스프레드시트 형태로 정리합니다. |
| `screenshot` | 사용자가 명시적으로 요청한 경우 데스크톱 또는 시스템 스크린샷을 캡처합니다. |
| `spreadsheet` | `.xlsx`, `.csv`, `.tsv` 같은 스프레드시트 파일을 생성, 수정, 분석, 포맷팅합니다. |

## 시스템 스킬

이 스킬들은 `.system` 아래에 있으며, 보통 내장 보조 워크플로우로 사용됩니다.

| 스킬 | 설명 |
| --- | --- |
| `openai-docs` | 최신 OpenAI 공식 문서와 제품/API 사용 가이드를 조회합니다. |
| `skill-creator` | Codex 스킬을 새로 만들거나 기존 스킬을 개선할 때 구조와 작성 방식을 안내합니다. |
| `skill-installer` | 큐레이션된 목록 또는 GitHub 저장소에서 Codex 스킬을 설치합니다. |

## 구조

각 스킬은 독립된 디렉터리로 관리되며, 일반적으로 `SKILL.md` 파일과 함께 필요한 참고 자료, 스크립트, 에셋을 포함합니다.
