# Codex Skills

This repository contains reusable Codex skills stored under `C:\Users\lenovo\.codex\skills`.

## Included Skills

| Skill | Description |
| --- | --- |
| `code-flowchart-mermaid` | Analyze code and draw Mermaid flowcharts for fragments, screen flows, navigation paths, lifecycle behavior, and core business logic branches. |
| `css-to-android-ui` | Convert CSS-driven UI definitions into Android View XML or Jetpack Compose code. |
| `doc` | Read, create, and edit `.docx` documents with layout-aware workflows. |
| `exofit-android-conventions` | Apply the exoFit repository's Android conventions, architecture boundaries, and delivery process. |
| `exofit-components-dev` | Develop and modify the Exofit React components library in line with its current structure and public APIs. |
| `figma` | Use the Figma MCP server to fetch design context, screenshots, variables, and assets from Figma. |
| `figma-implement-design` | Translate Figma nodes into production-ready code with close visual fidelity. |
| `gh-pr-korean` | Push the current branch and create or update GitHub pull requests in Korean, including template handling and Mermaid flowcharts. |
| `image-to-android-ui` | Recreate UI images, mockups, or screenshots as Android View XML or Jetpack Compose code. |
| `pdf` | Read, create, and review PDF files with rendering- and layout-aware workflows. |
| `screen-test-case-sheet` | Organize QA test cases into spreadsheet-style sheets grouped by screen units. |
| `screenshot` | Capture desktop or system screenshots when the user explicitly requests them. |
| `spreadsheet` | Create, edit, analyze, and format spreadsheet files such as `.xlsx`, `.csv`, and `.tsv`. |

## System Skills

These are stored under `.system` and are typically used as built-in helper workflows.

| Skill | Description |
| --- | --- |
| `openai-docs` | Retrieve up-to-date official OpenAI documentation and guidance for product and API usage. |
| `skill-creator` | Guide the creation or update of Codex skills with effective structure and workflows. |
| `skill-installer` | Install Codex skills from the curated list or from GitHub repositories. |

## Structure

Each skill is kept in its own directory and typically includes a `SKILL.md` file plus any supporting references, scripts, or assets needed by that workflow.
